# Comment #
## Constructor ##
```
Comment(coord_x, coord_y, text)
```
where:

> _coord\_x_ = x position of the box;

> _coord\_y_ = y position of the box;

> _text_ = defines content of the comment;

## Methods ##
### Edit ###
edits the content of the comment
```
edit(new_text)
```
where:
> new\_text = new content of the comment;
