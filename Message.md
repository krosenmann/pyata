# Message #
## Constructor ##
```
Message(coord_x, coord_y, text)
```
where:

> _coord\_x_ = x position of the box;

> _coord\_y_ = y position of the box;

> _text_ = defines content of the message;

## Methods ##
### Edit ###
edits the content of the message
```
edit(new_text)
```
where:
> new\_text = new text of the message;

### Click ###
bangs the message
```
click()
```
