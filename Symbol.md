# Symbol #
## Constructor ##
```
Symbol(coord_x, coord_y)
```
where:
> _coord\_x_ = x position of the box;

> _coord\_y_ = y position of the box;

## Methods ##
### Set ###
sets the Symbol to a specific value.
```
set(value)
```
where:
> _value_ = new value of the Symbol;

### Get Value ###
gets the value of the Symbol on Pd;
```
get_value()
```
